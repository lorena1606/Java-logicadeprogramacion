# Proyecto: Fundamentos de Java

Este proyecto demuestra los conceptos básicos del lenguaje Java mediante ejemplos prácticos organizados por temas.

## Tecnologías utilizadas

- Java
- Maven
- Visual Studio Code
- Git y GitHub

## Estructura del proyecto

El proyecto está organizado en clases que demuestran diferentes conceptos fundamentales de Java:

### 1. Variables y Constantes
- Declaración y asignación de variables
- Convenciones de nombres
- Uso de constantes con `final`

### 2. Tipos de Datos Primitivos
- byte, short, int, long
- float, double
- char
- boolean

### 3. Conversión de Tipos
- Casting implícito
- Casting explícito
- Pérdida de precisión

### 4. Operadores
- Aritméticos
- Relacionales
- Lógicos
- Asignación compuesta
- Operador ternario

### 5. Entrada de Datos
- Uso de `Scanner`
- Lectura de diferentes tipos
- Manejo del salto de línea pendiente

### 6. Estructuras Condicionales
- if
- if-else
- if-else-if
- switch

### 7. Estructuras de Repetición
- while
- do-while
- for
- for anidado
- break y continue

### 8. Métodos Estáticos
- Métodos sin parámetros
- Métodos con parámetros
- Métodos con retorno
- Sobrecarga de métodos

### 9. Arreglos Unidimensionales
- Declaración
- Inicialización
- Recorrido con for y for-each
- Uso de length

### 10. Arreglos Bidimensionales
- Declaración de matrices
- Recorrido de matrices
- Uso práctico de matrices

### 11. Manipulación de Cadenas
- Concatenación
- Métodos de String
- equalsIgnoreCase
- Inmutabilidad de String
- StringBuilder

### 12. Manejo Básico de Excepciones
- try-catch
- finally
- Manejo de errores comunes

## Ejecución del programa

El programa principal se encuentra en la clase:
App.java


Este archivo ejecuta todos los ejemplos de las diferentes clases del proyecto.

Para ejecutar el proyecto:

```bash
mvn compile
mvn exec:java
